# Job Salary Prediction Project

## Overview

This project focuses on analyzing employee and job-related data to understand the factors that influence salary trends and career growth. Using Exploratory Data Analysis (EDA), data visualization, and machine learning concepts, the project identifies important patterns related to experience, education, skills, and industry distribution.

The project provides meaningful insights into workforce trends and creates a strong foundation for future salary prediction models.

---

# Project Objectives

* Perform data cleaning and preprocessing
* Explore employee salary trends
* Analyze the impact of experience, skills, and education
* Visualize important patterns using graphs and charts
* Build a foundation for machine learning salary prediction

# Graph Insights

* Most employees belong to the medium experience category.
* Employees with more skills generally have better opportunities.
* Some job roles are more in demand than others.
* Higher education levels are common among employees.
* Certain industries employ more workers.
* Experience and skills show a positive relationship.
* Numerical variables show meaningful correlations.
